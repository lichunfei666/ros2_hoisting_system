# ROS2 Hoisting System Documentation

## 项目概述

ROS2 Hoisting System是一个基于ROS2的无人机吊装系统，用于实现无人机与地面设备之间的协同作业。该系统集成了DJI PSDK（Payload SDK），实现了对DJI无人机的控制，并通过通信模块实现与地面站的交互。

## 系统功能

- 无人机飞行控制
- 机器人运动控制
- 安全监控
- 视觉处理
- 地面站通信
- 负载设备控制
- 一键启动/停止服务
- Web界面监控
